ArrayLists

In this lesson you will learn how to use ArrayLists in Java.

Introduction

In Java, an ArrayList implements a List using an array. ArrayLists can only hold objects, not primitive types. Unlike arrays, ArrayLists are resizeable. ArrayLists keep track of both the capacity, and the current size of the array. The ArrayList class is found in the java.util package, so it must be imported at the top of your program like so:

import java.util.ArrayList;

Declaring an ArrayList

When declaring an ArrayList, you must include the type of object that will be stored in angle brackets as shown below.

ArrayList<String> words = new ArrayList<String>();
ArrayList<Integer> nums = new ArrayList<Integer>();

In Java Docs, the letter E is used to denote the type parameter. There are 2 constructors for ArrayLists.

//This will create an empty ArrayList of default capacity, which is 10
ArrayList<E>()

//This will create an empty ArrayList of the specified capacity
ArrayList<E>(int capacity)

Methods

The methods for the ArrayList class are as follows.

int size() //returns the size of the ArrayList
boolean isEmpty () //returns true if the ArrayList is empty, otherwise false
boolean add (E obj) //adds a new object to the ArrayList
void add (int i, E obj) //inserts the object obj at index i. i must be from - to size()
E set (int i, E obj) //sets index i to object obj and returns the set object
E get (int i) //returns the object at index i
E remove (int i) //removes the object at index i and returns the removed object
boolean contains (E obj) //returns true of obj is found in the array, otherwise false
int indexOf (E obj) //returns the index of obj if it is found, otherwise -1

Here is an example of how an ArrayList might be implemented.

ArrayList<String> cities = new ArrayList<String>();
cities.add("Toronto");
cities.add("Boston");
cities.add(0, "New York");
System.out.println(cities);

Which would output

[New York, Toronto, Boston]

The toString() method of ArrayList returns each element in the array seperated by commas, enclosed in [].

Using Primitive Types in an ArrayList

Now you might be thinking, But I thought you said you can't use primtive data types in ArrayLists? Well, you'd be correct. However, thanks to something called Autoboxing, primitive data types will automatically be converted to their wrapper classes when being added to an ArrayList.

ArrayList<Integer> nums = new ArrayList<Integer>();

nums.add(14);
nums.get(0); //returns 14

Review

What you need to know about ArrayLists:
  • The ArrayList class is found in the java.util package, so you must import that at the beginning of your program
  • ArrayLists are resizeable arrays
  • ArrayLists use the method size() to get the length of the array instead of the length property
  • ArrayLists are declared similar to an object, and must include the object type that will be stored in the ArrayList
  • You can use the add (E obj) method to add an element to the end of the array, and the add(int i, E obj) method to add an element at the specified index
  • You can use the get(int i) method to access an element in the array
