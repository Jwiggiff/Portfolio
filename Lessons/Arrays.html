Arrays

In this lesson you will learn how to use Arrays in Java.

Introduction

Arrays are a feature of the Java API that allow the programmer to store a list of values. arrays can store any primitive type or objects. In this lesson you will learn about how Arrays work, and how to use them in your programs.

Declaring an Array

First, we will learn about the basics of arrays. Arrays are lists of values that can be accessed by index. Here is an example of how you would define and initiate an array of ints in Java. 

int[] arr = new int[5];

As you can see, it is very similar to defining a regular variable. First, we have the type of the values that will be stored in the array, in this case int. Then we follow it with an opening and closing square brackets ([]) which tells Java that this is an array instead of just one int. Then we have the name of the variable, in this case arr. Next, is the equals sign and then new int[5]. This tells Java that we want to make a new int array of length 5. In Java, arrays are a fixed length, meaning that once we define the array it cannot be resized. Knowing all of this information, we can write the syntax to define an array in Java as this.

<type of variable>[] <variable name> = new <type of variable>[<length of array>];

Arrays can be of any primitive type or object. For example if you wanted to define an array of Strings, you'd do this.

String[] arr = new String[5];

The past examples were all examples of one-dimensional arrays. This means that the dimensions of the array are 1 by the length of the array. However, Java also allows you to have multi-dimensional arrays. Besides 1D arrays, the most common arrays are 2D and 3D arrays. Here are examples of defining 2D and 3D arrays.


          //2D int array
          int[][] arr = new int[5][5];
          //3D int array
          int[][][] arr = new int[5][5][5];

Initializing an Array

Initializing an array is putting values into an array. When you declare an array, each index is either initialized to 0, false or null depending on the data type. To set a specific index in an array you would do the following.

arr[0] = 4;

In this example, arr is the name of the array and 0 is the index. 4 is the value we are assigning to that index in the array. If we wanted to assign multiple indexes, you'd write it as follows.

//Sets value at index 0 to 4
          arr[0] = 4;
          //Sets value at index 2 to 6
          arr[2] = 6;
          //Sets value at index 1 to 32
          arr[1] = 32;

You can also intialize the array in the same line as you declare it.

int[] arr = {2, 4, 6, 8, 10};

If you wanted to initialize all indexes of an array, you could use a for loop to loop through each index in the array as follows.

for (int i = 0; i < arr.length; i++) {
            arr[i] = i;
          }

As you can see, we used a property of the array called length. This property can be used whenever you want to access the length of the array. However, remember that since arrays are not resizeable, once you declare an array, the length property is static.

Accessing an Array

As you may have noticed above, to access a specific index in an array, you use the name of the array followed by the index you want to access enclosed in square brackets.

System.out.println(arr[3]);

The above example will print out the value in the third index of the array. One important thing to note is that indexes in an array begin at 0. This means that the first value in an array is at index 0, not index 1.

Review

What you need to know about arrays:
  • Declared using the following syntax: 
    <type of variable>[] <variable name> = new <type of variable>[<length of array>];
  • initialized using one of three methods:
    • For a single element: 
      arr[0] = 6;
    • For all the elements: 
      for (int i = 0; i < arr.length; i++) {
                arr[i] = i;
              }
    • In the same line as declaration: 
      int[] arr = {2, 4, 6, 8};
  • To access the length of an array, use the length property
  • To access an index in an array, use the name of the array followed by the index you want to access enclosed in square brackets
